# Glosarium

## A
**Addendum Kontrak** — Perubahan thd kontrak existing (nilai/waktu/scope)
**Admin** — Role akses penuh
**APBD/APBN** — Sumber dana: Anggaran Pendapatan Belanja Daerah/Nasional
**ARUMANIS** — Air Minum & Sanitasi Cianjur
**Assign Pekerjaan** — Menugaskan user ke pekerjaan tertentu
**Audit Trail** — Log aktivitas pengguna

## B
**Berkas** — Dokumen digital (PDF, Word, Excel)
**Broadcast Notifikasi** — Kirim notifikasi massal

## D
**Dashboard** — Halaman utama ringkasan data
**Data Quality Stats** — Statistik kualitas data

## F
**Fiscal Year** — Tahun anggaran
**Foto** — Dokumentasi gambar pekerjaan

## I
**Impersonasi** — Admin login sebagai user lain

## K
**Kecamatan** — Wilayah adm di bawah kabupaten
**Kegiatan** — Program utama dengan pagu anggaran
**Kegiatan Role** — Batasi akses role ke kegiatan tertentu

## M
**Menu Permissions** — Visibilitas menu per role

## O
**Operator** — Role pengelola data operasional
**Output** — Hasil dari pekerjaan

## P
**Panel Pengawasan** — Aplikasi lapangan di `/pengawasan/` untuk role pengawas
**Pagu** — Jumlah anggaran (Rp)
**Pekerjaan** — Proyek turunan kegiatan
**Penerima Manfaat** — Individu/kelompok penerima manfaat program
**Penyedia** — Vendor/kontraktor
**Permission** — Izin spesifik untuk role
**Progress Estimasi** — Estimasi fisik/keuangan mingguan; sinkron Puspen ↔ panel pengawasan

## R
**RAB** — Rencana Anggaran Biaya
**RBAC** — Role-Based Access Control
**RKA** — Rencana Kegiatan dan Anggaran
**Role** — Peran pengguna (Admin/Operator/Viewer)
**Route Permission** — Aturan akses URL per method & role

## S
**Session** — Masa berlaku login
**Sidebar** — Menu navigasi kiri
**SPAM** — Sistem Penyediaan Air Minum
**SSO (Single Sign-On)** — Login sekali di Arumanis, otomatis masuk Panel Pengawasan via token handoff

## T
**Toast Notification** — Notifikasi pop-up sementara

## U
**User** — Pengguna aplikasi dengan akun login

## V
**Viewer** — Role baca saja
