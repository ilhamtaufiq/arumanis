# Komponen UI Dasar

## Button (Tombol)
Varian: Primary, Secondary, Destructive, Ghost, Outline, Link.
States: Default, Hover, Active, Disabled, Loading.

## Card
Container data ringkas. Widget dashboard, ringkasan, media card.

## Table
Sorting (klik header), pagination, row selection, striped, responsive scroll.

## Form Input
- **Text/Email/Number/Password** — Input standar
- **Select, Searchable Select, Async Searchable Select, Async Multi Select** — Dropdown
- **Textarea, Checkbox, Switch, Slider, Date Picker, Currency Input, Search Input**

## Modal / Dialog
- **Dialog** — Form/detail dalam modal
- **Alert Dialog** — Konfirmasi hapus/destruktif

## Sheet
Panel samping (kanan/kiri) untuk detail/form kompleks.

## Badge
Label status: Aktif (hijau), Nonaktif (merah), Notifikasi (angka).

## Breadcrumb
Navigasi hierarkis. Contoh: Master Data > Kecamatan > Detail

## Avatar
Foto profil dengan fallback inisial.

## Tooltip
Info saat hover elemen.

## Alert/Banner
Info (biru), Success (hijau), Warning (kuning), Error (merah).

## Skeleton
Animasi loading placeholder (table/card/list).

## Chart
Bar, Line, Pie, Area chart interaktif.

## Toast Notification
Pop-up pojok kanan atas. Success (~3s), Error (~5s), Info (~3s), Warning (~4s).

## Lainnya
Scroll Area, Collapsible, Popover, Separator, Progress, Tabs, Navigation Menu, Command Menu (Ctrl+K).
