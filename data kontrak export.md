1. Data Pekerjaan (Paket)
${nama_paket} : Nama Paket Pekerjaan
${pagu} : Nilai Pagu (format angka dengan pemisah ribuan, misal: 100.000.000)
${pagu_terbilang} : Pagu dalam bentuk teks (misal: Seratus Juta)
${kode_rekening} : Kode Rekening Pekerjaan
${kecamatan} : Nama Kecamatan lokasi pekerjaan
${desa} : Nama Desa lokasi pekerjaan
2. Data Program & Kegiatan
${nama_program} : Nama Program
${nama_kegiatan} : Nama Kegiatan
${sub_kegiatan} : Nama Sub-Kegiatan
${tahun} : Tahun Anggaran
3. Data Kontrak
${nilai_kontrak} : Nilai Kontrak (format angka dengan pemisah ribuan)
${nilai_kontrak_terbilang} : Nilai Kontrak dalam bentuk teks
${nomor_sppbj} : Nomor SPPBJ
${nomor_spk} : Nomor SPK
${nomor_spmk} : Nomor SPMK
${tgl_sppbj} : Tanggal SPPBJ (format lengkap Bahasa Indonesia, misal: 12 Januari 2024)
${tgl_spk} : Tanggal SPK (format lengkap)
${tgl_spmk} : Tanggal SPMK (format lengkap)
${tgl_selesai} : Tanggal Selesai Kontrak (format lengkap)
4. Data Penyedia (Kontraktor/Perusahaan)
${nama_penyedia} : Nama Perusahaan Penyedia (misal: CV. Maju Jaya)
${direktur} : Nama Direktur Perusahaan
${alamat_penyedia} : Alamat Perusahaan
${bank} : Nama Bank Perusahaan
${norek} : Nomor Rekening Perusahaan
${masa_hari} : Jumlah hari (misal: 90)
${masa_hari_terbilang} : Ejaan penyebutannya (misal: Sembilan Puluh)
${notaris} : Nama Notaris
${no_akta} : Nomor Akta Notaris
${tanggal_akta} : Tanggal Akta Notaris (format lengkap seperti 12 Januari 2024)
