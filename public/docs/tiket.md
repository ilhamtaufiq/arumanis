# Panduan Membuat Tiket

Halaman ini menjelaskan cara melaporkan kendala atau error yang Anda temui saat menggunakan aplikasi ARUMANIS.

---

## Kapan Harus Membuat Tiket?

Anda disarankan untuk membuat tiket jika mengalami salah satu kondisi berikut:

| Kondisi | Contoh |
|---------|--------|
| **Pekerjaan tidak muncul** | Anda sudah login, tetapi daftar pekerjaan di Dashboard kosong. |
| **Error pada aplikasi** | Muncul pesan error atau halaman tidak bisa dibuka. |
| **Data tidak sesuai** | Nama proyek, lokasi, atau nilai kontrak berbeda dari yang seharusnya. |
| **Gagal upload** | Foto atau berkas tidak berhasil diunggah meskipun sudah dicoba berulang kali. |

---

## Langkah-langkah Membuat Tiket

### 1. Buka Menu Tiket

Dari sidebar aplikasi, klik menu **Tiket** atau **Laporan Kendala**.

### 2. Klik Tombol "Buat Tiket Baru"

Di halaman daftar tiket, klik tombol **+ Buat Tiket Baru** di pojok kanan atas.

### 3. Isi Formulir Tiket

Lengkapi informasi berikut:

- **Judul Tiket**: Ringkasan singkat masalah (contoh: "Pekerjaan tidak muncul di Dashboard")
- **Kategori**: Pilih kategori yang sesuai:
  - `Bug / Error` - Untuk kesalahan teknis aplikasi
  - `Data Tidak Sesuai` - Untuk masalah data yang salah
  - `Akses / Permission` - Untuk masalah hak akses
  - `Lainnya` - Untuk kendala lain
- **Deskripsi**: Jelaskan masalah secara detail, termasuk:
  - Apa yang Anda coba lakukan
  - Apa yang terjadi (error message jika ada)
  - Kapan masalah terjadi
- **Screenshot** *(opsional)*: Lampirkan tangkapan layar jika memungkinkan

### 4. Kirim Tiket

Klik tombol **Kirim** untuk mengirimkan tiket Anda ke Admin.

---

## Contoh Tiket yang Baik

<div class="google-btn" style="display: block; padding: 16px; margin: 16px 0;">

## Status Tiket

Setelah tiket dikirim, Anda dapat memantau statusnya:

| Status | Keterangan |
|--------|------------|
| **Menunggu** | Tiket baru, belum ditinjau Admin |
| **Diproses** | Admin sedang menangani masalah |
| **Butuh Info** | Admin membutuhkan informasi tambahan dari Anda |
| **Selesai** | Masalah sudah diselesaikan |
| **Ditolak** | Tiket tidak valid atau duplikat |

---

## Tips Penting

> [!TIP]
> Semakin detail informasi yang Anda berikan, semakin cepat Admin dapat menyelesaikan masalah Anda.

> [!IMPORTANT]
> Jika masalah sangat mendesak dan mempengaruhi banyak pengguna, hubungi Admin langsung melalui kontak darurat yang tersedia.
