**Fitur:** CRUD unit SPAM, capaian SPM, pengelola, anggaran, impor data

## Tujuan

Modul SPAM Unit mengelola data unit Sarana Penyediaan Air Minum (SPAM) di setiap desa. Setiap unit mencatat kapasitas layanan, struktur pengelolaan, capaian sambungan rumah (SR) dan jiwa terlayani (SPM), serta rencana anggaran.

## Langkah Penggunaan

### Tambah Unit SPAM Baru

1. Klik tombol **Tambah Unit** (icon `+`)
2. Pilih **Kecamatan** -> **Desa** tempat unit berada
3. Isi **Nama Unit** (jika ada)
4. Atur toggle **Terdaftar SIMSPAM**
5. Isi kapasitas layanan (mata air, air tanah, lain-lain) dalam liter/detik
6. Isi data pengelola POKMAS jika tersedia
7. Klik **Simpan**

### Edit Unit SPAM

1. Klik ikon **Edit** pada baris unit
2. Ubah data yang diperlukan
3. Klik **Simpan**

### Hapus Unit SPAM

1. Klik ikon **Hapus**
2. Konfirmasi dialog

### Impor Data dari CSV/Excel

1. Klik tombol **Import** (icon Upload)
2. Pilih file dengan format sesuai template
3. Klik **Upload**
4. Sistem memproses dan menampilkan hasil

### Lihat Detail Unit

1. Klik ikon **Lihat**
2. Tab tersedia:

| Tab | Isi |
|-----|-----|
| **Info** | Data pokok unit, kapasitas layanan |
| **Pengelola** | Struktur POKMAS, Perdes |
| **Achievements** | Capaian SPM per tahun (SR, KK, Jiwa, BJP) |
| **Budgets** | Rencana anggaran, nilai kontrak, sumber dana |

### Tambah Capaian SPM (Achievement)

1. Buka detail unit -> tab **Achievements**
2. Klik **Tambah Achievement**
3. Isi: Tahun, Jumlah SR, KK, Jiwa, BJP KK, BJP Jiwa
4. Klik **Simpan**

### Tambah Anggaran (Budget)

1. Buka detail unit -> tab **Budgets**
2. Klik **Tambah Budget**
3. Isi: Tahun, Nilai Kontrak, Nama Paket, Sumber Dana
4. Klik **Simpan**

### Hapus Anggaran

1. Buka detail unit -> tab **Budgets**
2. Klik ikon **Hapus** pada baris anggaran
3. Konfirmasi

## Troubleshooting

| Masalah | Penyebab | Solusi |
|---------|----------|--------|
| Desa tidak muncul di dropdown | Kecamatan belum dipilih | Pilih kecamatan dulu |
| Gagal simpan unit | Field wajib kosong | Lengkapi data desa dan SIMSPAM |
| Gagal import | Format file tidak sesuai | Gunakan template CSV yang benar |
| Data tidak muncul setelah simpan | Belum refresh | Klik filter atau reload halaman |
| Tidak bisa menghapus unit | Data masih dirujuk | Hapus capaian/anggaran terkait dulu |
