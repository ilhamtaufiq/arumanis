## Kegunaan

Checklist memastikan item wajib per fase/pekerjaan terpenuhi (dokumen, lapangan, administrasi) sebelum tahap berikutnya.

## Alur kerja

1. Buka **Checklist**
2. Filter tahun / pekerjaan
3. Tandai item selesai / unggah bukti jika diminta
4. Simpan perubahan
5. Ekspor **Excel/PDF** dari aksi unduh (jika tersedia di halaman)

## Master fase

Admin dapat mengelola **Master Fase** (jika menu tampil) — template item checklist mengacu ke konfigurasi ini.

## Tips

- Jangan centang tanpa bukti — audit trail bisa dilacak
- Ekspor berkala untuk rapat evaluasi
