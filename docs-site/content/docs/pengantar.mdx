**ARUMANIS** (Air Minum & Sanitasi Cianjur) adalah sistem informasi untuk merencanakan, melaksanakan, mendokumentasikan, dan memantau program air minum & sanitasi.

Dokumentasi ini dibuka lewat menu **Panduan** di aplikasi (atau tautan di halaman login).

## Dua permukaan aplikasi

| Aplikasi | Siapa |
|----------|--------|
| **Arumanis Utama** | Admin, operator, PPTK, viewer |
| **Panel Pengawasan** | Pengawas / konsultan pengawas (SSO) |
| **Puspen** | Pusat pengendalian (KPI, progress, media) |

Semua memakai akun **APIAMIS** yang sama. Panel pengawas **tidak punya form login sendiri** — masuk lewat Arumanis.

<img src="/docs/assets/screenshots/landing-01-hero.png" alt="Landing Arumanis" className="rounded-lg border" />

## Mulai cepat

1. Buka aplikasi → [Login](/docs/auth)
2. Pahami [navigasi global](/docs/navigasi-global) (sidebar, notifikasi, profil)
3. Sesuai peran:
   - **Admin/operator** → [Dashboard](/docs/dashboard) → [Pekerjaan](/docs/pekerjaan-output)
   - **Pengawas** → otomatis diarahkan ke [Panel Pengawasan](/docs/pengawas-panel)
4. Unduh/preview dokumen → [OnlyOffice](/docs/dokumen-onlyoffice)
5. Masalah? → [Pemecahan masalah](/docs/pemecahan-masalah)
