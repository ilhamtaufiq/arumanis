---
title: Glosarium
description: Istilah yang sering muncul di Arumanis dan dokumen pengadaan.
---

| Istilah | Arti |
|---------|------|
| **ARUMANIS** | Air Minum & Sanitasi Cianjur — nama aplikasi |
| **APIAMIS** | Backend Laravel (API) pasangan frontend |
| **BFF** | Backend-for-Frontend di server Bun |
| **Pekerjaan** | Paket/proyek lapangan yang dikelola di sistem |
| **Kegiatan** | Program/kegiatan anggaran induk |
| **Output** | Komponen hasil pekerjaan (dasar matriks foto) |
| **Berkas** | Dokumen terlampir pada pekerjaan |
| **OnlyOffice** | Document Server untuk preview/edit office |
| **SIPD** | Sistem Informasi Pemerintahan Daerah (anggaran/Renja) |
| **SIPD Lite** | Layanan cache Renja/Penganggaran (`sipd-lite.cianjur.space`) |
| **Renja** | Rencana Kerja (sub kegiatan / penganggaran) |
| **SPSE** | Sistem Pengadaan Secara Elektronik (inaproc) |
| **Non-tender / PL** | Pengadaan langsung |
| **SPM** | Standar Pelayanan Minimal |
| **SPAM** | Sistem Penyediaan Air Minum |
| **Penerima** | Penerima manfaat (individu/komunal) |
| **Penyedia** | Vendor/kontraktor |
| **Pengawas** | Personil pengawasan lapangan |
| **Puspen** | Pusat pengendalian (modul monitoring) |
| **Tiket** | Catatan kendala/tindak lanjut |
| **SSO** | Single Sign-On (token/cookie lintas app) |
| **Impersonate** | Admin masuk sebagai user lain |
| **Media** | File tersimpan (Spatie Media Library) |
| **ZIP berkas** | Arsip unduhan semua dokumen pekerjaan |
