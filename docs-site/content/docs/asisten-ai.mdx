**Siapa:** user dengan menu AI / chat  

## Asisten AI

1. Buka **Asisten AI**
2. Ketik pertanyaan seputar data/prosedur aplikasi
3. Jawaban stream per token (tunggu hingga selesai)
4. Jangan kirim data sensitif di luar kebijakan organisasi

### Kapan berguna

- Mencari cara pakai modul
- Merangkum alur kerja
- Draft teks laporan (selalu diverifikasi manusia)

### Batasan

- Bukan pengganti data resmi di database
- Bisa salah — cek ke modul sumber
- Koneksi model/API bergantung konfigurasi server

## Chat internal

Jika menu **Chat** aktif: percakapan antar user di organisasi. Gunakan untuk koordinasi ringan; isu formal tetap lewat [Tiket](/docs/tiket).
