## Modul utama

| Halaman | Fungsi |
|---------|--------|
| Beranda Puspen | Pintu masuk modul |
| Progress fisik | Rekap progress lapangan |
| KPI pengawas | Kinerja pengawas |
| Review pekerjaan | Tinjauan paket |
| Media sharing | Bagikan media via token |
| Organize PDF | Susun/merge PDF |
| Sign PDF | Tanda tangan PDF |

## Media sharing

1. Buat share dari UI media sharing
2. Bagikan tautan token (publik terbatas)
3. Penerima buka tautan share tanpa login penuh (sesuai desain share)

## PDF tools

- **Organize** — urutkan, pilih halaman, unduh hasil
- **Sign** — bubuhkan tanda tangan pada PDF

Tools serupa juga bisa tersedia di menu Tools untuk role tertentu.

## Tips

- Gunakan filter tahun/anggaran agar rekap tidak overload
- Export/unduh file besar butuh sabar — gunakan unduhan browser native
