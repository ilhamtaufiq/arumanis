## Ringkasan

Panel Pengawasan adalah SPA lapangan (React + Bun BFF) untuk personil yang ditugaskan mengawasi paket pekerjaan. Data dan RBAC tetap di **APIAMIS**; panel hanya menampilkan **pekerjaan yang di-assign** ke akun login.

| Area | Fungsi |
|------|--------|
| Dashboard | KPI, paket perlu perhatian, filter tahun |
| Pekerjaan | Daftar + detail (6 tab) |
| Foto lapangan | Slot 0–100% per output, GPS/EXIF |
| Progress estimasi | Fisik & keuangan mingguan (sinkron Puspen) |
| Buat laporan | Laporan RAB rencana/realisasi per paket |
| Tiket | Isu lapangan + komentar |
| Notifikasi | Inbox & broadcast |
| Panduan in-app | `/pengawasan/panduan` |

```text
Arumanis (/sign-in) ──SSO/handoff──►  /pengawasan  (www/pengawas)
                                            │
                         cookie httpOnly    │
                         (pengawas_session) │
                                            ▼
                                       APIAMIS Laravel
```

## Masuk & sesi

### Alur utama (SSO dari Arumanis)

1. Login di **Arumanis** (`/sign-in`) dengan akun APIAMIS.
2. Jika role pengawas/konsultan pengawas **tanpa** role admin/manager, browser dialihkan ke panel (`/pengawasan/...` + token/handoff).
3. Panel menampilkan **Menyinkronkan sesi SSO…** lalu menghapus token dari URL.
4. Cookie **`pengawas_session`** (httpOnly) dipakai untuk request berikutnya.

Detail teknis SSO: [Autentikasi](/docs/auth#sso-ke-panel-pengawasan).

### Halaman login panel

| Path | Keterangan |
|------|------------|
| `/pengawasan/login` | Form login (email/password APIAMIS) bila dibuka langsung |
| `/pengawasan/sign-in` | Alias login |
| Tanpa sesi di rute terlindungi | Redirect ke Sign In Arumanis (+ path asal) |

### Logout & impersonate

| Aksi | Hasil |
|------|--------|
| **Keluar** di sidebar | Hapus sesi → Sign In Arumanis |
| **401** | Tombol **Masuk ulang** |
| **Impersonate** (admin dari Users Arumanis) | Banner kuning; Stop Impersonate kembali ke dashboard Arumanis |

## Dokumentasi foto

<img src="/docs/assets/screenshots/pengawas-foto.png" alt="Tab foto pengawas" className="rounded-lg border" />

### Slot wajib per output

**0% · 25% · 50% · 75% · 100%**

### Aturan

- **Individu** — foto per penerima yang ditugaskan; isi jiwa + NIK di tab Penerima
- **Komunal** — centang Komunal; minimal 5 foto per komponen (tanpa NIK/jiwa wajib)
- Status agregat paket: **Belum ada foto** / **Belum Selesai** / **Selesai**

### Cara unggah

1. Tab **Foto** → klik slot kosong
2. Koordinat: GPS otomatis, EXIF, atau **manual** (peta/input)
3. Pilih file → unggah
4. Preview / ganti / hapus bila perlu
5. **Cetak Foto** → PDF (izinkan popup browser)

Upload gagal di jaringan buruk: cek antrian unggah tertunda (banner) dan coba lagi.

## Progress estimasi

1. Tab **Progress** di detail pekerjaan
2. Pilih minggu aktif
3. Isi **Rencana** dan **Realisasi** per item
4. **Simpan** (aktif hanya jika ada perubahan)

Data tersinkron dengan **Puspen / rekap estimasi** di Arumanis.  
Paket **dibatalkan (canceled)** tidak perlu dilanjutkan di lapangan.

## FAQ

**Bagaimana login?**  
Lewat Arumanis `/sign-in`. SSO mengalihkan ke `/pengawasan`. Bisa juga login di `/pengawasan/login` jika diizinkan.

**Kenapa paket saya kosong?**  
Assignment user–pekerjaan belum diset di Arumanis (menu penugasan), atau filter tahun salah.

**Komunal vs individu?**  
Komunal = kelompok (tanpa jiwa/NIK). Individu = perorangan (wajib jiwa + NIK).

**Foto status “Belum Selesai”?**  
Belum semua slot 5 persen terisi per output, atau komunal belum mencapai minimal foto.

**Progress tidak bisa disimpan?**  
Tombol Simpan hanya aktif jika ada perubahan Rencana/Realisasi.

**GPS tidak muncul?**  
Izinkan lokasi browser, atau isi koordinat manual.

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Loop redirect setelah login | Build terbaru; hard-reload; login ulang dari Arumanis |
| Blank setelah deploy | Hard-reload / clear cache browser |
| 401 berulang | Masuk ulang via Arumanis; cek cookie domain |
| 403 | Paket di luar assignment — hubungi admin |
| GPS / upload gagal | Izin lokasi; file valid; coba manual koordinat |
| Popup cetak diblokir | Izinkan popup untuk domain Arumanis |
| Offline sementara | Antrian foto tertunda; unggah ulang saat online |

Lihat juga: [Pemecahan masalah](/docs/pemecahan-masalah)
